Monitoramento & Controle (M&C)

Redmine Plugin

To read this content in english click here

O Plugin M&C é utilizado como uma ferramenta gráfica de apoio para facilitar o acompanhamento das atividdes registrada em um projeto redmine.

Características

Versão atual (v0.1.0) - para Redmine 2.0.x

Gerencimento de Tarefas
  • Gráfico das tarefas agrupadas por Situação
  • Gráfico apresentando as tarefas gerenciáveis e não gerenciáveis
  • Tarefas gerenciáveis são aquelas com a data de entrega estabelecida e que ainda não foram para alguma situação do tipo fechada.
  • Tarefas não gerenciáveis são aquelas com a data de entrega não preenchida, ou seja, sem condições de estabelecer um parâmetro de mensuração.
  • Gráfico apresentando as tarefas atrasadas dentro do grupo das tarefas gerenciáveis.
Gerencimento de Tempo
  • Gráfico de linhas com horas planejadas e horas executada até as datas de entrega.
  • Gráfico de barra com o total de horas estimadas e o total de horas executadas
Gerencimento de Recursos Humanos
  • Gráfico de barras com o total de tarefas por responsável.
  • Gráfico de barras com o total de tarefas por responsável agrupado por situação.

Instalação

Pré-requisitos
  1. Redmine trunk version em http://redmine.rubyforge.org/svn
Via GitHub

  1. Copie ou clone a o conteúdo do plugin para a pasta de plugins do Redmine: /vendor/plugins/

    git clone http://github.com/alexmonteiro/Redmine-Monitoring-Controlling.git 
#{RAILS_ROOT}/plugins/redmine_monitoring_controlling

  2. Reinicie o Redmine e um novo módulo e permissões de acesso estarão disponíveis.

Arquivo ZIP ou TarGz
  1. Descompacte o arquivo .zip ou .targz do plugin na pasta de plugins do Redmine: /plugins/redmine_monitoring_controlling
  2. Reinicie o Redmine e um novo módulo e permissões de acesso estarão disponíveis.

Créditos

O Monitoramento & Controle foi criado baseado em javascript e rubyonrails:

  • Highcharts: biblioteca javascript para criação de gráficos;
  • Railscasts episódio 223: aprenda como montar gráficos em rails utilizando Highcharts.
